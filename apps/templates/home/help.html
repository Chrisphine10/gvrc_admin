Help & Documentation

Complete guide to using the GVRC Management System

Dashboard

The Dashboard is your main control center. Here you can:

  • View system overview and key metrics
  • See recent activities and updates
  • Access quick user management
  • Manage facilities and services
  • Access documents and content

Data Management

User Management

Manage system users and permissions:

  • Create new user accounts
  • Set user roles and permissions
  • Activate/deactivate accounts
  • Reset passwords
Facility Management

Manage facilities and their services:

  • Add new facilities
  • Set facility locations and coordinates
  • Configure available services
  • Manage facility contacts

Content Management

Documents

Organize and manage system documents:

  • Upload files and documents
  • Organize by categories and tags
  • Set access permissions
  • Search and retrieve files
Music Library

Manage audio content and playlists:

  • Upload audio files
  • Create and organize playlists
  • Add metadata and descriptions
  • Track usage analytics

How to Navigate

Sidebar Menu

Use the left sidebar to navigate between different sections:

  • Dashboard: Main overview page
  • Users: Manage user accounts
  • Facilities: Manage facilities
  • Documents: File management
  • Music: Audio library
Common Actions

Look for these buttons to perform actions:

  • Add New: Create new records
  • Edit: Modify existing records
  • Delete: Remove records
  • Search: Find specific records

Getting Started

First Steps
  1. Login with your credentials
  2. Explore the Dashboard overview
  3. Check your user profile and permissions
  4. Familiarize yourself with the sidebar menu
Need Help?

If you encounter issues:

  • Check this help page first
  • Contact your system administrator
  • Look for help icons (?) on pages
  • Use the search function to find features
